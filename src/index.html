Bienvenidos al Workshop de GitHub

Aprende a colaborar y gestionar tus proyectos de manera efectiva.

¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo que permite a los desarrolladores trabajar juntos en proyectos de software.

Ramas

Las ramas son versiones paralelas de tu proyecto. Puedes crear ramas para trabajar en nuevas características sin afectar la rama principal.

Pull Requests

Los pull requests son una forma de proponer cambios en un proyecto. Permiten revisar y discutir los cambios antes de fusionarlos en la rama principal.

Conceptos Básicos CAMBIOS

  • Repositorio:Acabo de agregar este cambio en mi branch personal Un espacio donde se almacena tu proyecto.
  • Commit:Agregando nueva rama/colab Un cambio guardado en el repositorio.
  • Merge:Colaboración de Salem Combinar cambios de diferentes ramas.
